# @algolens/database

This package provides a PocketBase-based database abstraction layer for AlgoLens, offering client-side database operations, authentication services, user management, problem and bookmark handling, with both browser and server-side support.

All packages in this monorepo use Bun as the JavaScript runtime.

For setup instructions, project structure details, and contribution guidelines, please refer to the main [README.md](../../README.md) in the root directory.
