# AlgoLens Backend

This package provides the main REST API server for AlgoLens using Hono framework, offering endpoints for problem management, step-by-step algorithm visualization, user authentication, and premium features integration.

All packages in this monorepo use Bun as the JavaScript runtime.

For setup instructions, project structure details, and contribution guidelines, please refer to the main [README.md](../../README.md) in the root directory.
