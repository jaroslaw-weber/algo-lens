# AlgoLens Frontend

The main frontend web application for AlgoLens, built with Astro and React, providing interactive algorithm visualization, problem exploration, user authentication, code editing with Monaco, and data visualization using Cytoscape and Chart.js.

All packages in this monorepo use Bun as the JavaScript runtime.

For setup instructions, project structure details, and contribution guidelines, please refer to the main [README.md](../../README.md) in the root directory.
